---
title: "Get started: install your first iTerm2 color scheme"
description: "Get your first iTerm2 color scheme installed in under two minutes using the GUI or the bash import script. Works with all 485+ schemes."
---

iTerm2 Color Schemes provides two paths to get a new theme running: a point-and-click GUI import and a bash script that can import one scheme or all 485+ in a single command. Both methods write directly to iTerm2's preferences — no configuration files to edit by hand.

## Before you start

Clone the repository to get all scheme files locally:

```bash
git clone https://github.com/mbadolato/iTerm2-Color-Schemes.git
cd iTerm2-Color-Schemes
```

All `.itermcolors` scheme files are in the `schemes/` directory.

## Choose your installation method

<Tabs>
  <Tab title="Keyboard shortcut (fastest)">
    <Steps>
      <Step title="Open iTerm2 profile settings">
        Launch iTerm2, then press **⌘+i** to open the current session's profile settings.
      </Step>
      <Step title="Navigate to Colors">
        Click the **Colors** tab in the profile settings panel.
      </Step>
      <Step title="Import a scheme">
        Click **Color Presets** → **Import**, then navigate to the `schemes/` folder in the cloned repository. Select one or more `.itermcolors` files.
      </Step>
      <Step title="Apply the scheme">
        Click **Color Presets** again and choose the scheme you just imported from the list.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Preferences menu">
    <Steps>
      <Step title="Open Preferences">
        In the menu bar, click **iTerm2** → **Preferences...** (or press **⌘+,**).
      </Step>
      <Step title="Open your profile">
        Select **Profiles**, then choose the profile you want to change.
      </Step>
      <Step title="Navigate to Colors">
        Click the **Colors** tab.
      </Step>
      <Step title="Import and apply">
        Click **Color Presets** → **Import**, select the `.itermcolors` file(s) from the `schemes/` directory, then click **Color Presets** again to activate the scheme.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Bash script">
    <Steps>
      <Step title="Run the import script">
        From the cloned repository root, run one of these commands:

        ```bash
        # Import all color schemes
        tools/import-scheme.sh schemes/*

        # Import all color schemes (verbose mode)
        tools/import-scheme.sh -v schemes/*

        # Import a specific scheme by file path
        tools/import-scheme.sh 'schemes/SpaceGray Eighties.itermcolors'

        # Import a specific scheme by name
        tools/import-scheme.sh 'SpaceGray Eighties'

        # Import multiple specific schemes
        tools/import-scheme.sh Molokai 'SpaceGray Eighties'
        ```

        <Warning>
          The bash script method is macOS-only. Quotation marks are required for scheme names that contain spaces.
        </Warning>
      </Step>
      <Step title="Restart iTerm2">
        Quit and reopen iTerm2 to reload the updated configuration file. The imported schemes will appear in **Color Presets**.
      </Step>
      <Step title="Apply the scheme">
        Press **⌘+i**, go to **Colors** → **Color Presets**, and select your newly imported scheme.
      </Step>
    </Steps>
  </Tab>
</Tabs>
