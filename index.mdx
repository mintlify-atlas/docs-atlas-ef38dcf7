iTerm2 Color Schemes is a community-maintained collection of over 485 beautiful color themes for iTerm2, with automatic ports to more than 20 terminals and editors. Whether you prefer dark moody palettes like Dracula and Tokyo Night or bright themes like Alabaster and GitHub Light, there's a scheme ready to install in minutes.

## What's included

Each color scheme is available in the native `.itermcolors` format for iTerm2, and is automatically exported to a wide range of other terminals and editors using the project's Python-based generation tooling.

## Get started

<Steps>
  <Step title="Browse the scheme gallery">
    Check the [dark themes](/schemes/dark-themes) and [light themes](/schemes/light-themes) pages, or jump straight to the [GitHub screenshots directory](https://github.com/mbadolato/iTerm2-Color-Schemes/tree/master/screenshots) to preview thumbnails for every scheme.
  </Step>
  <Step title="Download or clone the repository">
    ```bash
    git clone https://github.com/mbadolato/iTerm2-Color-Schemes.git
    cd iTerm2-Color-Schemes
    ```
  </Step>
  <Step title="Install in your terminal">
    Follow the [iTerm2 installation guide](/installation/iterm2) for macOS, or pick your terminal from the [Installation](/installation/other-terminals) section.
  </Step>
  <Step title="Use in Vim or VS Code">
    See [Vim schemes](/using/vim) or [VS Code schemes](/using/vscode) to activate your chosen theme in your editor as well.
  </Step>
</Steps>

<Note>
  All schemes are community-contributed. See [CREDITS.md](https://github.com/mbadolato/iTerm2-Color-Schemes/blob/master/CREDITS.md) for attribution of each theme to its original author.
</Note>
